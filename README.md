### 📱 Task Manager — Android (Kotlin + Room DB)
> Native Android productivity app with local persistence, reminders, and priority-based task sorting.

**Tech:** `Kotlin` `Android Studio` `Room DB` `Firebase` `Material Design 3`

**Highlights:**
- MVVM architecture with ViewModel + LiveData
- Offline-first: all data stored locally with Room; Firebase sync on connection
- Push notifications using WorkManager for task deadlines

### 🤖 Sentiment Analysis — Python ML Pipeline
> NLP model to classify product reviews as positive / negative / neutral using TF-IDF + Logistic Regression.

**Tech:** `Python` `Pandas` `Scikit-learn` `TensorFlow` `Jupyter Notebook` `Matplotlib`

**Highlights:**
- Preprocessed 10,000+ reviews with tokenisation, stop-word removal, and lemmatisation
- Achieved 91% accuracy on test split
- Deployed as a Flask REST API for integration with web apps

[![Repo](https://img.shields.io/badge/GitHub-View_Repo-181717?style=flat-square&logo=github)](https://github.com/Husainiya)
